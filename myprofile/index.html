Project Title: IBP-PepsiCo

  • IBP-PepsiCo manages producer and consumer goods data from various sources like Teradata and Azure Blob.
  • Initial data is stored in Azure Cloud Storage's landing layer.
  • Data goes through bronze and silver layers for cleaning and formatting using Azure Databricks with Pyspark transformations.
  • Enriched data is then loaded into the Processing layer for Data Science Team to perform ML operations.
  • Processed Dataframes are placed back in the silver layer.
  • Data is finally loaded into either the Gold or Synapse layer, serving as input for the Analytical Team.
  • Power BI is used to publish results to dashboards for visualization and analysis.
